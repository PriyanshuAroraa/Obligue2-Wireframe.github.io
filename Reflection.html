Summer 21´

Our goal with this page was to create a very clean and simple looking page that gave the user a “feeling” of summer, however this is not always easy. Here is some parts of the process and our thoughts around it.

When creating the wireframes, we first decided that we wanted a small preview of the photos on the top of the page under the header. Beneath the preview we wanted six sections that would contain the images as well text describing the picture. Creating the sections was the hardest part of the website. Common problem we ran into was getting the sections to be even and getting the images to fit inside the sections so that every image would be aligned with each other. Sizing the images to the correct size before using them was also a challenge. Some of the pictures had different dimensions from each other so we had to get them all to the same width and height, we did this on our phones before putting them into the html-file.

The styling and positioning of the text in each section was also quite difficult. The main issues we had was that the text tended to get “kicked” out of the div-box so working around that took some time. In addition to this when we put the text inside the div-box it came under the picture something we expected, but when trying to use the “float-right” tag we noticed that the text was unresponsive. We fixed this by using the “margin-right” tag.

When using the “@media only” tag it was quite difficult to style the different parts of the website so that they would reach the end goal that we had set. Especially the photos turned out to be quite challenging to style, sometimes they wouldn’t respond to the CSS-attributes we applied on them, and it was a struggle to position them as well. Then we used the “img-set” element which helped us design and change the size of the images that we used. We had already given the size measurements for the laptop page, and with “imgset” we could just implement the images and they would fit in the mobile page. As the obligue asked, we also changed the header and the paragraphs but that was a very easy job.
